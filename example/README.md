# Aplicación de Ejemplo del Sistema de Diseño E-commerce

## Descripción

Este proyecto es una aplicación de ejemplo diseñada para demostrar y previsualizar todos los componentes desarrollados en el paquete `ecommerce_sample_design_system`. Aquí podrás ver en acción los átomos, moléculas, organismos y fundamentos del sistema de diseño, organizados de acuerdo con la metodología Atomic Design.

## Cómo Ejecutar la Aplicación

Sigue estos pasos para poner en marcha la aplicación de ejemplo:

1.  **Navega al directorio del ejemplo:** Abre tu terminal y cambia el directorio a la carpeta `example/` dentro del proyecto principal:
    ```bash
    cd example
    ```

2.  **Instala las dependencias:** Asegúrate de tener todas las dependencias necesarias ejecutando:
    ```bash
    flutter pub get
    ```

3.  **Ejecuta la aplicación:** Inicia la aplicación en un emulador o dispositivo conectado:
    ```bash
    flutter run
    ```

La aplicación se abrirá, mostrando la pantalla principal ShowcaseHomePage.

## Cómo Navegar por los Componentes

La aplicación de ejemplo está organizada en diferentes pantallas, cada una dedicada a mostrar un tipo específico de componente:

*   **Fundamentos (Foundations)**: Aquí encontrarás ejemplos de cómo se aplican los colores, tipografías y espaciados definidos en el sistema de diseño.
*   **Átomos (Atoms)**: Demostración de los componentes más pequeños y reutilizables, como `AppButton`, `AppText`, `AppIcon`, etc.
*   **Moléculas (Molecules)**: Muestra combinaciones de átomos que forman componentes funcionales, como `AppSearchBar` o `ProductRating`.
*   **Organismos (Organisms)**: Presenta componentes más complejos, como `ProductCard`, `HorizontalProductList`, `CustomAppBar` y `CustomBottomNavBar`, que agrupan moléculas y átomos.

Desde la **Showcase Home Page**, puedes navegar a cada una de estas secciones para explorar los componentes individualmente.

## Componentes Mostrados

El ejemplo incluye showcases para:

*   Moléculas (`Molecules`)
